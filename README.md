Money Mission is a gamified financial literacy web app designed for kids aged 7–12. It teaches saving, spending, and money basics through fun missions, quizzes, and rewards.

Features:
	•	Duolingo-style learning through play
	•	Coins, stars, and rewards system
	•	Piggy bank savings goals
	•	Daily quizzes and puzzles
	•	Parent-safe design (no accounts or personal data)

Tech Stack:
	•	Python
	•	Streamlit
	•	Pandas
	•	NumPy
